# recivu-api-issues

---

This repository is the official channel for Recivu partners to report bugs and submit feedback related to the Recivu API.

**It contains no code.** It is used exclusively for issue tracking.

### How to open an issue

1. Go to the [**Issues**](../../issues) section
2. Click **New Issue**
3. Choose the appropriate template:
   - 🐛 **Bug Report** — to report unexpected API behavior
   - 💬 **Feedback** — for suggestions or general observations
4. Fill in all fields and submit

You will receive an email notification when the issue is updated or closed.

> For urgent matters, contact the Recivu team directly through your dedicated channels.
